# AI Smart Daily Planner

Application intelligente de gestion de tâches quotidiennes basée sur AWS et l’IA générative.

## Features

- AI task generation
- Smart prioritization
- Intelligent scheduling
- Productivity analytics
- Weekly AI reports
- Reminder notifications

## AWS Services Used

- AWS Lambda
- Amazon DynamoDB
- Amazon API Gateway
- Amazon S3
- Amazon SNS
- Amazon EventBridge
- Amazon CloudWatch
- AWS IAM
- PartyRock
